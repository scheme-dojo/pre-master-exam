# is-odd?

# question
- Create a function called is-odd? that takes a number as its argument and if it's an odd number return true, if it's not return false.
- If the definition is insufficient, please define it yourself.

# prerequisites

- (is-odd? number)

# example

```
(print (is-odd? 7)) 
; => #t

(print (is-odd? 2)) 
; => #f
```
